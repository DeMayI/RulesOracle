## Battleshock tests
If a unit lost one or more models, or has taken 3 or more wounds/hull points worth of damage this turn if it is a MONSTER or VEHICLE, its controlling player must roll 2d6. If the result is less than the units cool characteristic the test is failed and the unit becomes battleshocked. If the unit is below half strength(ie half its wounds/hull points or half of its starting models), subtract 1 from that test. If a unit starts the cleanup phase battleshocked, it must take a battleshock test this phase, even if it did not lose one or more models. 

## Remove Remaining Command Points
If either player has any unspent Command Points that they recieved at the start of the turn(Not counting command points that they started the battle with), they are lost. 

## Battleshock
While a unit is battleshocked, it has an OC characteristic of 0 and cannot be targeted with strategems. 

## Pinning
Some rules cause an enemy unit to be pinned. While an enemy unit is pinned, subtract 2 from its Movement Charactersitic and when that unit makes an attack, subtract 1 from the hit roll.
